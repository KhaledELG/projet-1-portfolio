# About me

> En tant que spécialiste en ingénierie Cloud et DevOps, je suis actuellement chez CGI, où j'ai dirigé des projets aux exigences stimulantes, couvrant le déploiement d'applications jusqu'à la création d'API sur AWS. Grâce à ma formation à Centrale Nantes, j'ai acquis les outils nécessaires pour aborder de manière méthodique les défis techniques, en adoptant une perspective globale. J'ai un fort attrait pour le travail collaboratif et je suis constamment avide d'acquérir de nouvelles connaissances. J'aime équilibrer les détails techniques avec une vision stratégique pour stimuler l'innovation dans les équipes Cloud.
